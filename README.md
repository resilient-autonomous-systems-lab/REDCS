# REDCS

### run it
- clone repo
```
git clone https://github.com/resilient-autonomous-systems-lab/REDCS.git
git checkout demo_linear_system_fcn
```
- run
```
run workshop.m
```

### Network structure:

- Generator: <br>

Relu, sigmoid, tanh, linear

- Discriminators: <br>

Relu, relu, relu, linear
